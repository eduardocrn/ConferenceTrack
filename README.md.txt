## Conference Track Management

### Prerequisites for Building
* Java 1.8 or above
* JUnit

### IDE
* Eclipse

### Build

* Using eclipse IDE
or
* Terminal
```
	javac -d bin src/app/*.java
```

## Runnig
 * Using Eclipse IDE
 or
 * Terminal
 ```
	java app.Main {StringFilePath}
	Ex: java app.Main "data/input.txt"
````
